== Description ==

**Cursul Valutar BNR** este un plugin care permite afisarea si calculul **cursului valutar** oficial, actualizat de pe pagina bancii nationale romane (BNR).
Plugin-ul poate fi personalizat pentru a se integra cat mai bine si in orice design de blog.
Acesta poate afisa **cursul valutar** la zi, cat si valori din zilele precedente.
Are de asemenea integrat un convertor valutar.
Pluginul afiseaza data ultimei actualizari si afiseaza mesaj de eroare in cazul in care aceasta nu a reusit.

English:
This plugin show exchange rate for RON from BNR (www.bnr.ro).

== Installation ==

Romana:

1. Se descarca fisierul, se dezarhiveaza si se copiaza continutul arhivei in directorul wp-content/cursul-valutar-bnr/plugins

2. Din meniul de administrare wordpress, optiunea Plugins se activeaza pluginul

3. Apoi Se merge in Appereance/Widgets si folosind "drag and drop" pluginul se trece in sectiunea sidebar

4. se efectueaza setarile, dimensiuni, culori, etc, apoi se salveaza.

5. In cazul in care aveti nelamuriri, probleme, sugestii, va rugam sa ne scrieti folosind formularul de contact, www.cursbnr.info/contact.php


English:

Instalation:

1. The download file, unzip and copy the directory wp-content/cursul-valutar-bnr/plugins

2. Wordpress administration menu, option activates plugin Plugins

3. Then go Appereance / Widgets and using "drag and drop" sidebar plugin pass in section

4. Settings are made, sizes, colors, etc., then save.

5. If you have any questions, problems, suggestions, please write to us using the contact form
